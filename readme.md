# Computer Science Quiz Platform

An interactive quiz platform for students from Classes 9 to 12.

## Features
- Chapter-wise quizzes
- Instant feedback and scoring
- Downloadable score cards
- Review answers with explanations
- Mobile responsive design

## Created by Gaurav Raj

### How to Use
1. Enter your name and select your class
2. Choose a chapter
3. Answer the questions
4. View your results and download score card

### Technologies Used
- HTML5
- CSS3
- JavaScript
- html2canvas for score card generation
